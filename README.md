# Dự án Xe Ô Tô Điện

Dự án mô phỏng, phân tích hoặc quản lý xe ô tô điện bằng Python.

## Cấu trúc dự án

- `file.py`: Logic chính
- `main.py`: Điểm khởi chạy
- `models.py`: Định nghĩa lớp đối tượng
- `utils.py`: Hàm tiện ích
- `requirements.txt`: Thư viện cần thiết
- `test_file.py`: Kiểm thử
- `data/`: Dữ liệu liên quan

## Hướng dẫn sử dụng

1. Cài đặt thư viện: 'pip install pandas numpy'
2. Chạy chương trình: `python main.py`
