# Dự án Xe Ô Tô Điện

Dự án mô phỏng đường đi của xe điện, dữ liệu từ ứng dụng trạm sạc EV

## Cấu trúc dự án

- `file.py`: Logic chính của chương trình
- `main.py`: Điểm khởi chạy ứng dụng
- `models.py`: Định nghĩa lớp đối tượng Car
- `utils.py`: Hàm tiện ích
- `requirements.txt`: Danh sách thư viện cần thiết
- `test_file.py`: Kiểm thử chức năng
- `data/`: Thư mục chứa dữ liệu liên quan (BOT.csv, charging_stations.csv, ...)

## Hướng dẫn sử dụng

1. Cài đặt thư viện: 'pip install pandas numpy'
2. Chạy chương trình: `python main.py`
